# 🚀 모노티 시작하기

모노티는 **웹, iOS, Android에서 사용할 수 있는 올인원 워크 & 라이프 플랫폼**입니다.  
소상공인과 고객을 연결하고, **일정 관리, 업무 관리, 빵장고, 렌티(예정)** 등의 서비스를 제공합니다.

> 📌 **이 문서에서는 모노티를 처음 시작하는 방법을 안내합니다.**

---

## 📲 1. 모노티 설치 방법

모노티는 **웹과 모바일 앱**에서 사용할 수 있습니다.  
아래에서 각 플랫폼별 설치 방법을 확인하세요.

<Tabs>
<TabItem value="web" label="💻 웹(Web)" default>

### **웹사이트에서 사용하기**
1. [모노티 공식 웹사이트](https://1472.ai/) 방문  
2. 우측 상단에서 **로그인** 클릭  
3. 회원가입 후 **바로 사용 가능!**  

</TabItem>

<TabItem value="ios" label="📱 iOS (아이폰)">

### **iPhone에서 설치하기**
1. [App Store](https://apps.apple.com/kr/app/%EB%AA%A8%EB%85%B8%ED%8B%B0-monoti-%EC%98%AC%EC%9D%B8%EC%9B%90-%EC%9B%8C%ED%81%AC-%EB%9D%BC%EC%9D%B4%ED%94%84/id6473604609) 방문  
2. **"다운로드"** 버튼 클릭  
3. 설치 후 **모노티 실행**  
4. 로그인 후 바로 사용 가능  

</TabItem>

<TabItem value="android" label="🤖 Android (안드로이드)">

### **Android에서 설치하기**
1. [Google Play](https://play.google.com/store/apps/details?id=com.notitle.toktokCalendar&pcampaignid=web_share) 방문  
2. **"설치"** 버튼 클릭  
3. 설치 완료 후 **앱 실행**  
4. 로그인 후 바로 사용 가능  

</TabItem>
</Tabs>

## 🔑 2. 회원가입 & 로그인

### **회원가입 절차**
1. 웹사이트 또는 앱 실행  
2. **이메일 및 비밀번호 입력**  
3. **가입 완료 후 로그인**  
4. **SNS 로그인 가능** (Google, Apple 등 지원 예정)

## 📋 3. 모노티 기본 기능

<Tabs>
<TabItem value="calendar" label="🗓 캘린더" default>

### **모노티의 핵심 기능 - 캘린더**
1. **개인 일정 / 업무 일정 / 모임 일정**을 나누어 관리 가능  
2. **친구, 연인과 일정 공유 기능 제공**  
   - 일정별 사진 업로드 가능 → 일정별 추억 기록  
   - 일정에 댓글을 남겨 소통 가능  
   - 일정 완료 후 **리포트 제공**  

</TabItem>

<TabItem value="bread" label="🍞 빵장고">

### **성심당과 함께하는 "빵장고"**
1. **모노티 앱에서만 사용 가능**  
   - 현재는 **대전 지역**의 성심당과 협업 중  
2. **보관 유형:** 캐리어 / 상온 / 냉장 보관 가능  
3. **예약 기능 지원** + **현장 결제 가능**  

</TabItem>

<TabItem value="renti" label="📦 렌티">

### **렌티 - P2P 단기 렌탈 플랫폼 (출시 예정)**
1. **P2P 렌탈 모델**: 개인 간 직접 대여 가능  
2. **단기 렌탈 특화**: 최대 1주일 단위로 운영  
3. **위치 기반 서비스**: 내 주변 이웃과 쉽게 물건 대여  

> 곧 출시될 **렌티 서비스**를 기대해주세요! 🚀  

</TabItem>
</Tabs>

## 🛠 4. 모노티 API 사용 예제 (준비 중)

### **모노티 API를 사용하여 주문 데이터를 가져오는 예제**
```js
fetch('https://api.monoti.com/orders')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));
```
